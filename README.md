# ⚖️ FairAI: The Forensic Ethics Lab

> **"Because fairness shouldn't be a black box."**

## 📖 The Story: Why FairAI?
Meet **Sarah**, a Head of Talent at a fast-growing tech startup. She recently implemented a "State-of-the-Art" AI tool to screen thousands of resumes. On paper, efficiency skyrocketed. But six months later, Sarah noticed a disturbing trend: **92% of the hired candidates were men.** 

The AI hadn't learned to find the "best" candidates; it had learned to find people who *looked* like previous hires, inadvertently penalizing women for career gaps or different educational backgrounds. Sarah had no way to "look inside" the AI to prove it—until now.

**FairAI** was born from the need to give people like Sarah a forensic laboratory to audit, expose, and fix these hidden "neural stereotypes" before they cause real-world harm.

---

## 🚀 The Solution
FairAI is a **Unified Forensic Workstation** that provides:
1.  **Tabular Bias Scanning:** A live ML pipeline that trains real models on production data (Hiring, Finance, Loans) to detect selection gaps in real-time.
2.  **Multi-Neural Probing:** A direct uplink to Hugging Face weights to audit the "ethical alignment" of LLMs like Gemma, Llama 3, and Mistral.
3.  **Explainability Suite:** Interactive correlation heatmaps and feature importance metrics that turn abstract "bias" into hard, actionable data.

---

## 🛠️ How We Built It (Implementation)
*   **Engine:** Python-based forensics using **Scikit-Learn** for live model training and **Pandas** for high-speed data manipulation.
*   **Intelligence:** Integrated **Hugging Face Inference API** to query and audit global LLM weights without requiring local GPU clusters.
*   **UI/UX:** A high-fidelity **Streamlit** dashboard designed with a "Titan Black x Gold" industrial aesthetic for enterprise-grade oversight.
*   **Entropy Engine:** A custom randomization logic using system timestamps to ensure every audit is a fresh, non-static "stress test."

## ⚙️ Setup & Usage
1. **Clone the Repo:** `git clone https://github.com/your-username/fairai`
2. **Install Dependencies:** `pip install -r requirements.txt`
3. **Set your API Key:** Create a `.env` file and add `HF_TOKEN=your_token_here`.
4. **Launch the Lab:** 
   ```bash
   streamlit run main.py
   ```
